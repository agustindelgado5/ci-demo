# CI Demo con GitHub Actions

## Descripción

Este proyecto demuestra la implementación de un pipeline de Integración Continua (CI) utilizando GitHub Actions.

## Funcionamiento

Cada vez que se realiza un push al repositorio, se ejecuta automáticamente un pipeline que:

- Restaura dependencias
- Compila el proyecto
- Ejecuta la aplicación

## Tecnologías utilizadas

- C#
- .NET
- GitHub Actions
